Detailed Description

Struct used to pass array(s) of options when creating a CoAP message.

This struct is intended to be used to hold on-stack initializers for message options.

struct GG_CoapMessageOptionParam* GG_CoapMessageOptionParam::next

This field allows options to be put in a linked list It may be set to NULL when options are passed as an array.

◆ sorted_next

struct GG_CoapMessageOptionParam* GG_CoapMessageOptionParam::sorted_next

This field allows options to be sorted. It is used internally by the CoAP runtime when serializing messages to datagrams, and should be ignored by users of the API. It is part of this structure for efficiency, so that the library does not have to allocate its own memory when sorting options.
