Detailed Description

Interface implemented by frame assemblers. A frame assembler is an object that can re-assemble a stream of bytes into a frame, typically for the purpose of re-assembling packets in packet-oriented protocols, when the data is received over a transport channel that doesn't already provide support for framing.
